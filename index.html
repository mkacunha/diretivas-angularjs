Utilizando minha diretiva ModalMsg

Modal pequena Modal média Modal grande

Para este exemplo foi utilizado a diretiva <modal-msg> e seus atributos title, message e size, onde:

  • title: foi utilizado para mostrar o título da modal, sendo que o mesmo também poderia ficar em branco. Mas neste exemplo foi utilizado para mostrar o título Atenção, meu título...;
  • message: atributo utilizado para mostrar a mensagem ao usuário. Neste exemplo foi utilizado para mostrar a mensagem Minha mensagem ao usuário...;
  • size: este atributo pode ser utilizado para configurar o tamanho da modal, sendo que neste exemplo foi utilizado o valor sm para modal pequena, lg para a modal grande, e para a modal média simplesmente não foi informado, sendo assim o tamanho default da modal.

Para este exemplo foi utilizado a diretiva <modal-msg> e seus atributos title, message, size, function_ok e function_cancel onde:

  • title: foi utilizado para mostrar o título da modal: Atenção...;
  • message: foi utilizado para mostrar a mensagem Realmente deseja remover o veículo {veiculo} ?;
  • function_ok: este atributo pode ser utilizado para passar por parâmetro qual função que deverá ser executada quando o usuário escolher a opção OK da modal, desta forma assim que a modal é fechada a função passada por parâmetro é executada. Neste exemplo foi utilizada a função remover(veiculo) que está localizada no controller ExemploModalMessageController;
  • function_cancel: tem o mesmo comportamento que o atributo function_ok, mas este é executado quando o usuário selecionar a opção cancelar da modal. Neste exemplo foi utilizada a função cancelou(veiculo) que está localizada no controller ExemploModalMessageController
